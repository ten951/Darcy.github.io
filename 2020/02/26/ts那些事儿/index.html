基础类型

  1. 布尔值 let isDone: boolean = false;
  2. 数字
  1. 字符串 let name: string = "bob"; 单双引号都行 并且以${ expr }这种形式嵌入表达式
  2. 数字 let list: number[] = [1, 2, 3];或者let list: Array<number> = [1, 2, 3]; 两种形式都行
  3. 元组 Tuple let x: [string, number];
  4. 枚举

默认情况是从0开始编号. 可以改成从1开始编号

  1. 任意值 any let notSure: any = 4; notSure = "maybe a string instead"; notSure = false; // okay, definitely a boolean

  2. 空值 void 代表没有返回值

  3. Null 和 Undefined 是所有类型的子类. so 可以把他们赋值给任意类型

  4. Never 那些永不存在的值的类型

  5. 类型断言 通过类型断言这种方式可以告诉编译器，“相信我，我知道自己在干什么”。 类型断言好比其它语言里的类型转换，但是不进行特殊的数据检查和解构。 它没有运行时的影响，只是在编译阶段起作用。 TypeScript会假设你，程序员，已经进行了必须的检查。
    let someValue: any = "this is a string"; let strLength: number = (someValue as string).length;
