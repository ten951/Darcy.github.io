shell命令

基本的bash shell命令

遍历文件

cd 路径名 用以指定想切换到的目录名. 如果没有指定目标路径名. 则切换到当期用户的主目录
pwd 可以显示出shell会话的当前目录，这个目录被称为当前工作目录。

文件和目录列表

ls -alf 显示当前目录下的文件和目录 默认是按照字母排序的 -F 可以区分文件和目录 -A 显示隐藏文件 -l 长列表格式输出

可以依据名称过滤. ?匹配一个字符 *多个字符

ls -l scri?t

处理文件

  1. 创建文件 touch
  2. 复制文件 cp source destination 将source复制到destination. -i选项可以提示是否要覆盖 -r会递归的将目录下的子文件复制过去.
  3. 链接文件 ln 分符号链接 -s 硬链接
  4. 重命名文件 mv 可以将文件或者目录移动到另一个位置或重新命名 移动文件不会影响文件的inode和时间戳 -i可以提示是否要覆盖
  5. 删除文件 rm -i 提示 -f 强制删除 -rf是非常暴力的
  6. 创建目录 mkdir -p 可以创建多级目录
  7. 删除目录 rmdir 只能删除空目录. 结合rm命令使用
  8. 查看文件类型 file
  9. 查看整个文件 cat -n是给所有行添加行号. -b 只会给文本添加行号 more 分页查看整个文件 less是more的升级版本
  10. 查看部分文件 tail 从尾部查看文件. 默认10行. -n可以指定行数 -f 它允许你在其他进程使用该文件时查看文件 head 从头部查看文件

检测程序

ps 命令 -e 显示所有运行在系统上的进程 -f 显示拓展输出 -l 更多信息长格式输出.它只能显示某个特定时间点的信息。如果想观察那些频繁换进换出的内存的进程趋势，用ps命令就不方便了。

top 命令 实时检测进程.
